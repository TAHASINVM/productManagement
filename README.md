# Steps 
    1. Clone the project
    2. cd project-name
    3. composer install
    4. create .env file ( cp .env.example .env)
    5. php artisan key:generate
    6. configure Database
    6. php artisan migrate
    7. php artisan serve
    8. You can then access and test the API using the URL "/docs/api"
